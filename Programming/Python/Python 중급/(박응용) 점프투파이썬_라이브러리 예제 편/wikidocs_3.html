전자책(E-Book) 구매

아래의 양식을 작성하신 후 결제하시면, 점프 투 파이썬 전자책(PDF)을 메일로 발송해드립니다.
(이메일이 도착하지 않은 경우 스팸함도 확인해 주세요)

서 명 : 점프 투 파이썬
가 격 : 6,000 원

※ 이메일 주소는 반드시 정확하게 입력해 주세요. 전자책은 결제 후 이메일을 통해 1시간 내로 전달됩니다.

※ 전자책은 이메일을 통해 다운로드 받으신 후 바로 이용이 가능하며 보유기간은 무제한입니다.

※ 문의 : pahkey@gmail.com

※ 전화번호는 전자책의 압축을 해제하기 위한 비밀번호로 사용됩니다.

※ 단순 변심, 상품하자의 경우 전자책이 포함된 이메일 수령일로부터 7일 이내까지 환불이 가능합니다.
※ 단순 변심인 경우에는 전자책의 특성 상 이메일에 포함된 전자책을 다운로드한 경우에는 환불이 불가합니다.
