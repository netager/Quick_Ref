## 02. 예외를 처리한다는 것은 무엇을 말하는 것이고 어디에서 처리해야 하는가? 		
	- 오류처리의 주체
		- 성적관리프로그램 --- API(Write() {권한, 파일존재, 파일용량 등}) --- 하드디스크
			- API에 문제가 발생하면 성적관리프로그램에 보고하면 성적관리프로그램에서 오류처리해야 하지만 리턴값을 주지 않는 경우도 발생
	- 자바에서 예외를 보고(통지)하는 값과 명령어 
		- throw new 권한없음 예외();
		- 	try {
				Write(...);
			}
			catch(권한없음예외 ex) {
				System.out.println("죄송");
			}
