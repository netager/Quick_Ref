## 콤비네이션 연산자
### Combinators
- Descendant selectors          : A B   : A의 자손들 중에 B를 찾아라
- Child selector                : A > B : A의 자식 중에 B를 찾아라  
- Adjacent sibling selector     : A + B : A의 이웃한 것의 B를 찾아라
- General sibling selector      : A ~ B : A의 이웃한 것들 중에서 B를 찾아라
